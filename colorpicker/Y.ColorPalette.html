Class Y.ColorPalette

ColorPalette provides a simple interface for storing and reusing selected colors. It can be coupled with the ColorPicker widget to provide a simple history of selected colors.

Properties

colors - Array

An array of colors stored in the palette
Default Value: []

Methods

addColor

void addColor ( color )
Adds the specified color to the palette if it doesn't already exist.
Parameters:
color <Object> the color to add, in HSL float object format e.g. { h: 1.0, s: 1.0, l: 0.5 }

colorExists

Boolean colorExists ( color )
Returns true if the specified color (in HSL float object format) already exists in the palette
Parameters:
color <Object> the color to test, in HSL float object format e.g. { h: 1.0, s: 1.0, l: 0.5 }
Returns: Boolean
true if the color already exists in the palette

onSwatchClick

protected void onSwatchClick ( ev )
Handler for the moment when a swatch in the palette is clicked. This handler fires the 'palette:selected' event.
Parameters:
ev <Object> the click event.

Events

palette:selected

palette:selected ( color )
An event fired when a color in the palette is clicked upon.
Parameters:
color <Object> An object containing the color in RGB and HSL format e.g. { rgb: { r: 255, g: 0, b: 0 }, hsl: { h: 1.0, s: 0.5, l: 1.0 } }
