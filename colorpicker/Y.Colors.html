Class Y.Colors

The Colors class provides simple color conversion functionality. It currently supports HSL to RGB conversions and vice versa. The conversion methods are used directly, e.g. Y.Colors.hslToRGB(1.0,1.0,1.0);

protected void _hue2rgb ( )
Internal method used for converting for HSL to RGB.

hslToRGB

object hslToRGB ( h , s , l )
Converts HSL values in the range 0-1 to RGB format in the range 0-255.
Parameters:
h <Number> hue component of the color in the range 0-1
s <Number> saturation component of the color in the range 0-1
l <Number> lightness component of the color in the range 0-1
Returns: object
the color in RGB format, returned as an object of the form { r: 255, g: 0, b: 0 }. The RGB values are normalised in the range 0-255.

rgbToHSL

object rgbToHSL ( r , g , b )
Converts RGB values in the range 0-255 to HSL format in the range 0-1.
Parameters:
r <Number> red component of the color in the range 0-255
g <Number> green component of the color in the range 0-255
b <Number> blue component of the color in the range 0-255
Returns: object
the color in HSL format, returned as an object of the form { h: 1.0, s: 0.5, l: 0.5 }. The HSL values are normalised in the range 0-1.
